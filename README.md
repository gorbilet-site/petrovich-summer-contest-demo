# Где лето лучше: в городе или на даче?

Демо-лендинг конкурса для Петровича и Горбилета.

## Что есть в демо

- split-screen с городом и дачей;
- выбор стороны участника;
- загрузка одного фото с локальным предпросмотром;
- переходы в два MAX-бота конкурса;
- ручная демо-отметка, что участник нажал «Участвую» в каждом боте;
- локальная демо-лента участников;
- счетчик города и дачи;
- фильтр ленты: все, город, дача;
- блок призов и промокода `КУЛЬТУРНЫЙ ДАЧНИК`.

## Важно

Это визуальный прототип. Фото и контакты в демо никуда не отправляются и не сохраняются на сервере.

Для боевого запуска нужен backend: прием заявок, приватное хранение фото, база участников, модерация и проверка участия через MAX-ботов.
